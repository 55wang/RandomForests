DecisionTree module¶

class TreeMethods.DecisionTree.DecisionTree(max_depth=2, min_size=1)[source]¶

Bases: object

A decision tree base class.

Classification and Regression Trees will be derived class that override certain functions of this class. This was done because many common methods, so to reduce code they are written here in the base class.

Atrributes:

max_depth (int): The maximum depth of tree.

min_size (int): The minimum number of datapoints in terminal nodes.

n_features (int): The number of features to be used in splitting.

root (dictionary): The root of the decision tree.

columns (list) : The feature names.
